### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 三、功能描述
系统分为（身份）：普通用户、店工、店主、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 门店信息管理
6. 订单信息管理



**店工功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 门店信息管理
5. 订单信息管理
6. 工作日志管理




**店主功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 员工管理
5. 门店信息管理
6. 门店员工管理
7. 工作日志管理




**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 顾客管理
4. 员工管理
5. 店主管理
6. 门店信息管理
7. 门店员工管理
8. 部门分类管理
9. 订单信息管理
10. 工作日志管理
